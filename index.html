FLIGHT MANUAL — CONTROLS

MOUSE / ↑↓←→
Pitch & roll — stick sense: ↓ / pull back = nose up. Y toggles the mouse-stick; then click the screen to capture the cursor and fly with relative movement (trackpad-friendly), ESC releases
TOUCH
Phones & tablets get a thumb stick + button deck automatically: left thumb flies, FIRE / AB / WPN / TGT under the right thumb, a drag-to-set throttle lever on the left, GEAR / HOOK / BRK / VIEW / MAP / pause along the top
F1 – F10
Throttle 10% – 100%  ·  F10 twice = afterburner
W / S
Throttle up / down  ·  SHIFT (hold) = afterburner
, / .
Rudder left / right
SPACE
Fire selected weapon (hold for the gun)  ·  MOUSE BTN = gun trigger
ENTER / TAB / 1 2 3
Select weapon (AMRAAM / Sidewinder / Gun) — voice callout confirms; 1/2/3 jump straight to one
T
Cycle radar target
R
Radar range (2 / 10 / 40 NM)
B
Wheel brakes / airbrake
G
Landing gear
H
Arrestor hook (F/A-18 only — the F-16 has none)
C / F
Chaff / Flares
M
ECM jammer toggle
D
Smoke trail toggle
KEYPAD
Point of view: 4/6 yaw · 8/2 pitch · +/− distance
SHIFT+P
Drop rescue pod (Mission 4)
V
Camera: cockpit / chase / orbit / tower
N
Live map toggle (position, carrier, airfields, bandits)
+ / −
Magnify view in / out (XMAG) — every view
SHIFT+E
Eject  ·  P / ESC pause  ·  ? this screen  ·  Q quit to menu
